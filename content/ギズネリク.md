'''ギズネリク'''市 （[[ガフィーク通商語]]：'''Qiznelik''' [ŋiznelik] ）は、[[ガフィーク都市同盟]]中部、[[ガフィーク都市同盟の地方行政区分#三界|人の国]]の[[ガフィーク都市同盟の地方行政区分#市の一覧|市]]。ガフィークでは最上位の自治体を慣例的に「市」と称するが、面積や規模としては他国の州や県に近い。[[環大洋]]に面し、[[地名未設定|地名未設定湾]]に位置する。[[レスカル共和国]]の学術都市[[アリコス]]とは海上国境を挟んで隣接しており、経済・学術の両面で盛んに交流がある。市翼（郊外）まで含めたギズネリク市全体の人口は約194万人。中世以来の歴史を持つ'''[[ギズネリク大学]]'''を擁し、世界各国からの留学生や研究者が集まる学術都市として知られる。
また歴史的経緯から、国内の最高司法機関である'''[[市嘴|ヴェルグゾップ'''（大市嘴）]]'''の所在地でもある。

== 地名 ==
ギズネリク（'''Qiznelik'''）という名称は '''iznelix''' 「知る人たち」<ref>古形。現代では語頭の i- がなく znelixとなる</ref>に場所名詞を作る接辞 '''q(a)- -Ç'''<ref>-Ç は語末の子音が[[ガフィーク語#子音浮揚|浮揚]]することを示す</ref> を付したもので「知る人の場所」あるいは「'''ズネリ'''（後述）の場所」を意味する。
'''zneli'''「知る人」 という語彙は、文字通りの意味以上にギズネリク大学における職位の名称（「教授」と訳されることが多い）として知られ、この意味合いでは歴史的に関連のある諸外国語でもそのまま「ズネリ」と呼ばれる場合がある。
また口語的にはギズネリク市域の住民一般、とりわけ大学関係者全体を（本来の「知の人々」の意味合いを込めて）「ズネリ」と称するケースがある。肯定的なケースでは「知的な人々」のニュアンスが含まれるが、「浮世離れした学者」のような否定的な用いられ方も散見される。

== 歴史 ==
現ガフィーク領の諸都市のうち、最も早い時期にその存在が記録に残されたのがこのギズネリクの地である。[[レスカル半島]]を中心に陸路で西進した[[ゼムアカ人]]の一団は急峻な[[名称未定峠01]]に止められるまで数百年にわたって開拓の手を進めることとなるが、この峠にほど近い入植地で最大のものがこの地であった。

他の[[ゼムアカ人]]の諸入植地と同様にこの土地も放牧と漁労を中心とした集落から出発していたが、峠越えという大きな目標に対して各地から知恵者が集まる独特の風土が記録に残されている。<ref>舟小屋通りのウフラドゥ『ゆく風の書』</ref> 元々ガフィークの移民社会は高い教育を受けた層が一定数おり、国としての基盤ができる以前から教育を重んじる気風はあったとされている。

レスカル人の侵攻の際に[[レスカル半島]]側から避難してきた住民たちの中には博識な老人や教育者など知的階級が多かったと伝えられる。これは当時の社会規範が知を尊ぶものであったため知識階級を先に逃がしたとも、単に知識階級がうまく逃げおおせたためとも言われる。

[[アリコスの断崖戦争]]の末にレスカル人との睨み合いがひと段落すると、ギズネリクを中心とする[[ゼムアカ人]]と西方の[[モルカ人]]はついに盟約を結ぶこととなった。この時に互いの持てる知識を共有し合うに及び、支配領域すべてから知恵者が終結していた[[ゼムアカ人]]側の知の集積は（モルカ人にとって新奇な内容が多かったことも手伝って）目覚ましく、「'''ギズネリクに万知あり'''（Qizneliki znaz fsua.）」「'''ガフィークには百の湖、ギズネリクには百の賢者'''（Qafiki astu izelix, Qizneliki astu iznelix.）」などとその学究都市的性格を言及されることとなる。

以降、国土東端の中心都市として、また巨大な学術都市として栄華を極めていくこととなる。

== 地理 ==
ギズネリク市は[[アリコス湾]]に面した港湾都市であり、湾を挟んで[[レスカル|レスカル共和国]]の[[アリコス|アリコス市]]と面している。[[アリコス|アリコス市]]もギズネリク同様に学術都市として知られており、両都市は姉妹都市として提携している。

== 区分 ==
== 気候 ==
[[コスマスの気候区分]]における[[亜寒帯湿潤気候]]の '''Dfb'''（ライ麦気候）にあたる。[[ガフィーク]]の他の多くの都市と同様、極めて寒冷で長い冬と冷涼で短い夏を特徴とする。夏季の気温は20℃を超えることは稀で、夏でも夕方には長袖が必要である。

== 教育 ==
{{main|ギズネリク大学}}
都市の顔とも言える[[ギズネリク大学]]の他、歴史的に教育に高い価値を見いだしてきた風土も手伝って初等～中等教育の充実度も国内随一である。とりわけ[[ギズネリク大学]][[キャオリーク・カレッジ]]の付属学校群は国内屈指の名門校でもあるため、首都出身の子女も寮や下宿住まいで中等部からギズネリク市で教育を受けることも多い。中等部の例ほど一般的ではないが、子女の教育のため家族ぐるみで初等部から転居する例もある。
